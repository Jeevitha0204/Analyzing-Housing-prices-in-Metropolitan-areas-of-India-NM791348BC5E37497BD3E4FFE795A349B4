# 🏠 Analyzing Housing Prices in Metropolitan Areas of India

## 📖 Project Description

This project analyzes housing prices across major metropolitan cities in India using data visualization and business intelligence techniques. The goal is to identify pricing patterns, understand factors influencing property values, and provide meaningful insights for investors, homebuyers, real estate developers, and policymakers.

The analysis is performed using Tableau dashboards and visual storytelling to transform raw housing data into actionable insights.

---

## 🎯 Objectives

- Analyze housing price trends in metropolitan cities of India.
- Study the impact of location, area, amenities, and infrastructure on property prices.
- Create interactive dashboards for better decision-making.
- Visualize housing market patterns and relationships between different variables.
- Support investment planning and real estate market analysis.

---

## 🏙️ Metropolitan Cities Covered

The project includes housing data from the following cities:

- Bangalore
- Chennai
- Delhi
- Hyderabad
- Kolkata
- Mumbai

## 🔍 Key Insights

- Housing prices vary significantly across metropolitan cities.
- Location plays a critical role in determining property value.
- Properties with better infrastructure and amenities generally command higher prices.
- Access to schools, hospitals, and maintenance services positively impacts housing demand.
- Urban growth and economic development contribute to rising property prices.
